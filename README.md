## 📌 Overview
A lightweight, high-performance system monitoring tool designed for cybersecurity investigations and resource optimization. This utility provides real-time visibility into system processes, memory allocation, and network sockets, specifically tuned to identify anomalous behavior.

## 🚀 Features
- **Live Process Tracking:** Monitor CPU, Memory, and Disk I/O with precision.
- **Network Forensics:** View active TCP/UDP connections and associated PIDs.
- **Anomaly Detection:** Flags unsigned processes or those running from temporary directories.
- **Optimized Performance:** Uses mathematical optimization principles to predict resource exhaustion.
